# Demostraciones Técnicas

Este directorio está reservado para almacenar videos y scripts de demostración interactiva.

## Plan de Grabación

### 1. Sistema HFT (Alta Frecuencia)
- **Script**: `demos/hft_demo.py` (simular feed de mercado)
- **Visualización**: Terminal split (Java Logs vs Rust Metrics)
- **Objetivo**: Mostrar latencia < 1ms en tiempo real.

### 2. IA Backend (Cold Start)
- **Script**: `curl` requests a la API.
- **Comparativa**: Docker Container (Python) vs Rust Process.
- **Objetivo**: Mostrar arranque instantáneo de Rust.

### 3. Sistema Operativo (Scheduling)
- **Script**: Ejecutar 100 procesos ligeros.
- **Visualización**: Gráfica de uso de CPU en ASCII art.
- **Objetivo**: Demostrar eficiencia del scheduler custom.
